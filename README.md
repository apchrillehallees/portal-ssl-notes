# portal-ssl-notes

本仓库用于归档和发布多个独立 HTML 页面，主要存放与 SSL/TLS 相关的一些笔记、参考或辅助页面。

## 项目简介

这是一个以静态 HTML 文件为主的仓库，每个页面独立存在，彼此之间无依赖关系。页面内容可能涉及 SSL 证书、TLS 协议、加密通信等主题的笔记或说明，适合作为个人参考或学习资料存档。

仓库本身不提供任何服务，也不针对特定网站或域名，仅作内容归档与展示之用。

## 目录说明

```
portal-ssl-notes/
├── index.html          # 首页/导航页（可选）
├── page-1.html        # 独立页面示例
├── page-2.html
└── ...
```

每个 HTML 文件均自包含，无外部资源依赖（除标准 CDN 或公共库外）。页面可直接在浏览器中打开，或通过 GitHub Pages 等静态托管服务发布。

## 页面归档说明

- 所有页面均为独立归档，不依赖后端或数据库。
- 页面内容可能包含个人笔记、技术总结或参考资料。
- 部分页面可能引用公开标准文档或 RFC，版权归原作者所有。
- 归档内容不保证实时更新，仅供参考。

## 维护说明

- 本仓库不定期更新，新增页面或修正内容。
- 欢迎通过 Issues 提交建议或错误反馈。
- 不接受针对具体网站或商业用途的修改请求。
